En esta ocasión explico brevemente como conectar audifonos (headset) por medio de bluetooth en el gestor de ventanas i3wm (para los que usamos este excelente gestor de ventanas sabemos que es bastante minimalista, no consume casi recursos y a cambio de eso hay que hacer algunas cosas a pie de vez en cuando y aprendemos) de cualquier manera este pequeño tutorial les puede servir a todos aquellos que usen una distro derivada de Debian con otros escritorios que no tengan el Blueman instalado y no les muestre el muy útil task-icon o i3bar-icon para enlazar dispositivos bluetooth en sus sistema.

Lo primero que tenemos que hacer es asegurarnos de que nuestro driver para el bluetooth esté correctamente instalado; Con los siguientes comandos instalaremos los paquetes necesarios con apt-get :

[sourcecode language="bash"]# Si fuese necesario purgar una instalación previa hay que correr este comando, si es la primera vez que se instala no es necesario:
$ sudo apt-get purge blueman bluez-utils bluez bluetooth[/sourcecode]

[sourcecode language="bash"]# Instalamos los paquetes con blueman en nuestro sistema
$ sudo apt-get install blueman bluez-utils bluez bluetooth[/sourcecode]

Antes de hacer uso de blueman tenemos que tener corriendo el daemon para el bluetooth con un comando parecido a esto:

[sourcecode language="bash"]# La manera clásica de correr el daemon de bluetooth:
$ /etc/init.d/bluetooth start
# Para las distros que usan systemctl para levantar y detener servicios:
$ systemctl start bluetooth.service[/sourcecode]

Después de esto utilizamos el lanzador de i3wm con MainKey+d o bien desde la terminal para ejecutar el asistente de blueman llamado blueman-assistant.

Una vez ejecutado el asistente es muy sencillo enlazar cualquier dispositivo siguiendo los pasos, es por ello que es un asistente y nos facilita la vida con botón de next -> next como se muestra a continuación en las imágenes:

[gallery ids="3069,3072,3070,3073" type="slideshow"]

Recordemos que podemos incluso entrar al asistente desde el bar-icon de nuestra barra de i3wm

Cuando tenemos enlazado nuestro dispositivo (headset) solo resta asignarlo como dispositivo de audio para el sistema, para lograr esto podemos utilizar una control llamado pavucontrol:

[sourcecode language="bash"]# Para instalar pavucontrol
$ apt-get install pavucontrol[/sourcecode]

En la imagen anterior he señalado con rojo el selector para decirle al sistema que dirija el audio de firefox a los headset Rapoo H6000, que son los que he enlazado.
